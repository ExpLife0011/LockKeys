#LockKeys#

##What is LockKeys##
LockKeys是widnows下的一个explorer扩展。在任务栏上显示大小写锁定和数字键盘状态。

##Why I create the LockKeys?
当我在使用GNOME的时候，我遇到了一款GNOME扩展[Lock Keys](https://extensions.gnome.org/extension/36/lock-keys/)。  
作为一个程序员，经常需要按下Capslock来切换大小状态。以及我使用的是108键键盘，也需要用到Numlock来切换小键盘状态。但是我在Windows平台下只能搜到使用Notify_icon，也就是托盘来显示的程序。  
于是我也想自己做一个类似的在Windows下的扩展。也就有了LockKeys。

##How to use the LockKeys?##
1. 下载一个release版本。
2. 双击realease包中的register.bat。(它可能需要UAC权限。)
3. 右键点击任务栏。 任务栏 -> LockKeys。你就可以看到LockKeys显示在任务栏。(你可能在第一次看不到LockKeys。你需要重新右键点击一次任务栏，应该就可以看到了。)
